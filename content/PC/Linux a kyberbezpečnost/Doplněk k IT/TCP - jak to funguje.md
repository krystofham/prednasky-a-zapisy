# TCP in a nutshell

Transportní protocol v TCP/IP modelu. Používá three-way-handshake.
## Three way handshake
Navázání spojení
1. SYN
	- Packet s příznakem SYN, chce navázat spojení
2. SYN-ACK
	- Server odpoví paketem SYN + ACK, souhlasí a potvrzuje
3. ACK
	- Klient pošle ACK a spojení je navázáno.
