# Podmínky a loopy
## Podmínky
Základní podmínka:
```
if (podminka) {
  // kód se provede, pokud je podmínka true
}
// else if
if (vek >= 18) {
  console.log("Jsi plnoletý");
} else {
  console.log("Nejsi plnoletý");
}
// else if
if (znamka === 1) {
  console.log("Výborně");
} else if (znamka === 2) {
  console.log("Chvalitebně");
} else {
  console.log("Další známka");
}
```
## For loop
```
for (let i = 0; i < 5; i++) {
  console.log(i);
}
```
## While loop
```
let i = 0;
while (i < 5) {
  console.log(i);
  i++;
}
```
