# Proměnné

Proměnná 
``` let PROMĚNNÁ = VALUE; ```
Konstanta 
``` const Konstanta = VALUE; ```
## Typy proměnných
### Základní (primitivní) typy
Číslo 
```let cislo = 10;        // number```
Float
```let float = 10.5;        // float```
Text
```let text = "Ahoj";     // string```
Boolean (pravda / nepravda) 
```let jeHotovo = true;   // boolean```
Pouze true a false. low key
Nedefinovaná hodnota 
```let promenna;          // undefined```
Prázdná hodnota
```let prazdna = null;    // null```
### Složené typy
Pole
```let pole = [1, 2, 3];  // array```
Objekt 
```
let osoba = {
  jmeno: "Jan",
  vek: 25
};                     // object
```
Zjištění typu ```console.log(typeof text);```
