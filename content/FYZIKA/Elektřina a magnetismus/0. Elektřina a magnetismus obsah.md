## 1. Elektrostatika
[[1. Elektrostatika]]

| Vzorec                                                             | Název                         |
| ------------------------------------------------------------------ | ----------------------------- |
| $\vec{F} = \frac{1}{4\pi\varepsilon_0} \frac{q_1 q_2}{r^2}\hat{r}$ | Coulombův zákon               |
| $\vec{E} = \frac{1}{4\pi\varepsilon_0} \frac{Q}{r^2}\hat{r}$       | Pole bodového náboje          |
| $\oint \vec{E} \cdot d\vec{A} = Q/\varepsilon_0$                   | Gaussův zákon (integrální)    |
| $\nabla \cdot \vec{E} = \rho/\varepsilon_0$                        | Gaussův zákon (diferenciální) |
| $\vec{E} = -\nabla\varphi$                                         | Pole jako gradient potenciálu |
| $\nabla^2 \varphi = -\rho/\varepsilon_0$                           | Poissonova rovnice            |
| $u_E = \frac{\varepsilon_0}{2}$                                    | $\vec{E}$                     |

## 2. Elektrické obvody
[[2. Elektrické obvody – R, L, C]]

|Vzorec|Název|
|---|---|
|$U = RI$|Ohmův zákon|
|$U = L,dI/dt$|Napětí na cívce|
|$I = C,dU/dt$|Proud kondenzátorem|
|$\sum I_k = 0$|Kirchhoff – uzlový zákon|
|$\sum U_k = 0$|Kirchhoff – smyčkový zákon|
|$\tau = RC$ nebo $L/R$|Časová konstanta|
|$\omega_0 = 1/\sqrt{LC}$|Rezonanční frekvence|
|$Z = R + j\omega L + 1/(j\omega C)$|Impedance RLC|

## 3. Magnetismus a indukce
[[3. Magnetismus a elektromagnetická indukce]]

|Vzorec|Název|
|---|---|
|$\vec{F} = q(\vec{E} + \vec{v}\times\vec{B})$|Lorentzova síla|
|$d\vec{B} = \frac{\mu_0}{4\pi}\frac{I,d\vec{l}\times\hat{r}}{r^2}$|Biot-Savartův zákon|
|$\oint \vec{B}\cdot d\vec{l} = \mu_0 I$|Ampérův zákon|
|$\nabla \times \vec{B} = \mu_0\vec{J}$|Ampérův zákon (dif.)|
|$\nabla \cdot \vec{B} = 0$|Neexistence monopolů|
|$\mathcal{E} = -d\Phi_B/dt$|Faradayův zákon|
|$\nabla \times \vec{E} = -\partial\vec{B}/\partial t$|Faradayův zákon (dif.)|
|$u_B =|\vec{B}|

## 4. Maxwellovy rovnice (vakuum)
[[4. Maxwellovy rovnice a elektromagnetické vlny]]
$$\nabla \cdot \vec{E} = \frac{\rho}{\varepsilon_0}, \qquad \nabla \cdot \vec{B} = 0$$

$$\nabla \times \vec{E} = -\frac{\partial \vec{B}}{\partial t}, \qquad \nabla \times \vec{B} = \mu_0\vec{J} + \mu_0\varepsilon_0\frac{\partial \vec{E}}{\partial t}$$

|Vzorec|Název|
|---|---|
|$c = 1/\sqrt{\mu_0\varepsilon_0}$|Rychlost světla|
|$\nabla^2\vec{E} = \mu_0\varepsilon_0,\partial^2\vec{E}/\partial t^2$|Vlnová rovnice|
|$\vec{S} = \vec{E}\times\vec{B}/\mu_0$|Poyntingův vektor|

## Matematické struktury – přehled

|Operátor|Definice|Použití|
|---|---|---|
|Gradient $\nabla f$|$(\partial_x f, \partial_y f, \partial_z f)$|$\vec{E} = -\nabla\varphi$|
|Divergence $\nabla \cdot \vec{F}$|$\partial_x F_x + \partial_y F_y + \partial_z F_z$|Gaussův zákon|
|Rotace $\nabla \times \vec{F}$|viz determinantový zápis|Ampérův, Faradayův zákon|
|Laplacián $\nabla^2 f$|$\partial_x^2 f + \partial_y^2 f + \partial_z^2 f$|Poissonova, vlnová rovnice|
